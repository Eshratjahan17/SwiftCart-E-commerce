#### 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?
map() is used to created a new array by changing each item of existing array. 

ForEach() is also used to loop in each item but it doesn't return a new array , map() is used to return a new array. 

#### 3) What is the difference between `==` and `===`?
`==` compares value only ,it converts type automatically.
`===` Compares value and type both.
